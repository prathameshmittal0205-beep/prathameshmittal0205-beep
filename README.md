### 📱 Offline Facial Recognition & Liveness Detection System
<p>
  <img src="https://img.shields.io/badge/React_Native-20232A?style=flat-square&logo=react&logoColor=61DAFB" />
  <img src="https://img.shields.io/badge/OpenCV-5C3EE8?style=flat-square&logo=opencv&logoColor=white" />
  <img src="https://img.shields.io/badge/Edge_AI-7AA2F7?style=flat-square" />
  <img src="https://img.shields.io/badge/Lightweight_CNNs-BB9AF7?style=flat-square" />
  <img src="https://img.shields.io/badge/On--device_ML-F7768E?style=flat-square" />
</p>
A fully offline, lightweight facial recognition and liveness detection system for secure authentication of field personnel in zero-network environments. *(Hackathon Project)*
- Designed an edge-deployable AI model (~20MB target size) for mid-range Android (8.0+) and iOS (12+) devices with <1 second inference time and no GPU dependency.
- Implemented anti-spoofing liveness detection using real-time user actions (blink, smile, head movement) to prevent photo/screen-based fraud.
- Integrated the AI pipeline with a React Native cross-platform mobile application, ensuring seamless offline authentication.
- Built local data storage with a sync-and-purge mechanism to securely upload records to the cloud (AWS-ready) once connectivity is restored.
- Achieved >95% recognition accuracy, maintaining robust performance under varied lighting conditions and diverse demographics.

<br />

### 💳 CreditLens AI – Explainable Credit Scoring
<p>
  <img src="https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white" />
  <img src="https://img.shields.io/badge/Scikit--learn-F7931E?style=flat-square&logo=scikitlearn&logoColor=white" />
  <img src="https://img.shields.io/badge/XGBoost-7AA2F7?style=flat-square" />
  <img src="https://img.shields.io/badge/Data_Clustering-BB9AF7?style=flat-square" />
  <img src="https://img.shields.io/badge/SHAP-F7768E?style=flat-square" />
</p>
An AI-powered credit scoring system evaluating the creditworthiness of gig workers, freelancers, and informal earners lacking traditional credit history. *(Hackathon Project)*
- Developed a behavior-based scoring pipeline using transaction-level data (income flow, spending patterns, financial stability).
- Implemented KMeans clustering to segment users into behavioral financial groups and used BG/NBD probabilistic modeling to estimate user financial activity and lifetime value.
- Trained an XGBoost classification model to predict credit risk and repayment probability with high accuracy.
- Integrated the SHAP explainability framework to provide transparent, human-readable reasons for every credit decision.
